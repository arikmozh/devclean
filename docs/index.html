Features

Safe, fast, and fully in your control

🔒

Safe by default

Scan is always read-only. Use --dry-run to preview exactly what will be deleted before committing.

📅

Age-based filtering

Use --older-than to only clean items older than N days. Protect active projects automatically.

🎯

Selective targets

Clean only what you want. Flags like --docker, --node-modules, --brew let you pick and choose.

⚡

Zero config

Install, run, done. No config files, no setup wizards, no accounts needed.

🚀

Fast scanning

Scans your entire home directory in seconds with smart traversal. Skips .git and .Trash.

🌐

Cross-platform

Works on macOS and Linux. Detects available tools and adapts automatically.
